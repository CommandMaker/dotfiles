# Dotfiles

Ce repository contient mes `dotfiles` en cas de formatage de mon **ArchLinux**.

## Thème actuel : catppuccin
[Leur Github](https://github.com/catppuccin/catppuccin)

## Installation des dotfiles

Le script `install` va vous permettre de réaliser l'installation de tous les fichiers du repo. Il peut **installer des paquets** ArchLinux (situés dans le fichier `.packagelist`, modifiable si vous ne voulez pas certains paquets) et **éditer** votre  `.config`. Sauvegardez donc vos configurations actuelles.
